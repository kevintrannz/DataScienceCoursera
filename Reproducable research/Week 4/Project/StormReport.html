Synopsis

In this assignment a data set was used to calculate what impact certain weather events had on the population health and economy in the US between 1950 and 2011. The data was mostly processed using the R package called dPlyr. Using this package the total fatalities, injuries, property damage and crop damage per event type were calculated. Fatalities and injuries were used to answer the question what events types was the most harmful with respect to population health. Property damage and crop damage were combined and then used to answer the question which events had the greatest economic consequences.

As a result it was concluded that across the US, tornadoes, excessive heat, and flash floods are most harmful with respect to population health. The second conclusion was that tornadoes, thunderstorm winds, and flash floods have the greatest economic consequences.

The data used was taken from the National Weather Service Instruction 10-1605.

Creating function that checks the inserted char for certain values, which then assigns the corresponding multiplier. This function is created because the crop and property damage columns had multipliers which had to be converted to calculate a total amount of crop & property damage per event type. The function returns the converted value in billions.

multiplyConv <- function(value, char) {
    multiply <- 1
    
    if(char == "H") {
        multiply <- 100
    }
    else if (char == "K") {
        multiply <- 1000
    }
    else if (char == "M") {
        multiply <- 1000000
    }
    else if (char == "B") {
        multiply <- 1000000000
    }
    
    return(value * multiply / 1000000000)
}

Note: The steps below could have been done in one step. However the result of that would be a huge chain of actions which would make it unclear for the reader to understand.

Convert the data to a new variable (dataN short for dataNeeded) and transform all the exponential values to uppercase so that it can be used in the function described above (this was done so that the function only has to read letters like M, instead of m and M)

dataN <- data %>% mutate(PROPDMGEXP =  toupper(PROPDMGEXP), CROPDMGEXP =  toupper(CROPDMGEXP))

Calculate the property & crop damage using the MultiplyConv function that is described above

dataN <- dataN %>% rowwise() %>% mutate(Propdmg = multiplyConv(PROPDMG, PROPDMGEXP), CropDmg = multiplyConv(CROPDMG, CROPDMGEXP))

Select columns that are needed for the following calculations.

dataN <- dataN %>% select(EVTYPE, FATALITIES, INJURIES, Propdmg, CropDmg)

Calculate the total amount of fatalities per event type and arrange them descending (so that the ones with the highest amount of fatalities are at the top of the list). After that take the first ten values of that list and put them in a new variable so that we now have the ten event types with the highest fatalities.

sumFata <- dataN %>% group_by(EVTYPE) %>% summarize(Fatalities = sum(FATALITIES)) %>% arrange(desc(Fatalities))
## Warning: Grouping rowwise data frame strips rowwise nature
sumFataTop <- sumFata[1:10,]

Calculate the total amount of injuries per event type and arrange them descending (so that the ones with the highest amount of injuries are at the top of the list). After that take the first ten values of that list and put them in a new variable so that we now have the ten event types with the highest injuries

sumInj <- dataN %>% group_by(EVTYPE) %>% summarize(Injuries = sum(INJURIES)) %>% arrange(desc(Injuries))
## Warning: Grouping rowwise data frame strips rowwise nature
sumInjTop <- sumInj[1:10,]

Calculate the total amount of economical damage per event type by summarizing the total property & crop damage and arranging them in descending order (so that the ones with the highest amount of economical damage are at the top of the list). After that take the first ten values of that list and put them in a new variable so that we now have the ten event types with the highest economical damage.

sumEcoDmg <- dataN %>% group_by(EVTYPE) %>%  summarize(EcoDmg = sum(CropDmg) + sum(Propdmg)) %>% arrange(desc(EcoDmg))
## Warning: Grouping rowwise data frame strips rowwise nature
sumEcoTop <- sumEcoDmg[1:10,]
